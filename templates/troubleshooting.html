Troubleshooting

MediaCrush relies on a lot of new technologies to get you videos. For the majority of our users, this isn't a problem. However, some users have trouble playing our videos. If videos aren't playing well for you, it might be one of the following problems:

  • Your web browser is not up-to-date
    • If you're on IE, we suggest switching to an alternative, or updating to at least IE 11.
  • Your system is missing codecs
    • On Windows, try installing VLC. It installs a lot of useful codecs.
    • On Linux, install all the ffmpeg browser packages for your selected web browser.
      • Using Firefox? Install all the gstreamer packages and search for "gstreamer" in about:config to enable it.
      • Using Chromium? Install ffmpeg-chromium-extra (it might be different on your system).
    • On Mac, VLC is your best bet, too.
  • On mobile? Try using the latest Chrome or Firefox for your device instead of the built-in browser.
